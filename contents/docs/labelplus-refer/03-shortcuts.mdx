# 快捷键列表

**窗口中**

* Alt+Q/W/E/R键：四个模式切换
* Alt+T: 紧急回避快捷键

**图片浏览界面中**

* Ctrl+鼠标左键：标号
* Ctrl+鼠标右键：删除所指标号
* 鼠标中键：向后翻页
* 鼠标悬停在标签上：显示对应文本
* 数字键1-9：更改添加标签的默认分组
* Q/W/E/R键：四个模式切换
* T键：隐藏窗口
* A：快速插入预置文本
* V：临时隐藏标号
* C：临时显示标号分组
* 键盘←/→：翻页
* Tab：向后翻页

**文字输入界面中**

* Ctrl+回车：输入下一项文本
* Ctrl+上下：切换编辑的标签项目
* Ctrl+左右：翻页
* Alt+A：快速输入短语菜单

**标签列表中**

* 数字键1-9：更改选中项的分组
* Del键：删除选中标签
